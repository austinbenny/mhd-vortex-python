\subsection{Installation and Usage (Linux / macOS)}

To reproduce the results, it is recommended to extract the package into a dedicated directory to ensure all dependencies and build scripts are correctly located.

\begin{minted}{bash}
mkdir hw
unzip package.zip -d hw

cd hw

curl -LsSf https://astral.sh/uv/install.sh | sh

uv sync

make
\end{minted}

\subsection{Usage}
